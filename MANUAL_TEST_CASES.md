# 手動テストケース

このドキュメントは、実データを使って次の一連フローを確認するための手順です。

- 通常手紙を受信する
- 受信した相手へ「一冊の本を送る」
- 相手名でその一冊を受け取る

## 前提

- ブラウザで `index.html` を開いていること
- 初期データが残っていても問題ありません
- 必要に応じて運営室で「デモデータを初期化」して開始してください

## テストデータ

- 受信者A（最初の利用者）: 夜の読書人A
- 受信者B（相手側の受信名）: 窓辺の受取人B
- 一冊の本の送信者名: 夜の読書人A
- 一冊の本の宛先名: 窓辺の受取人B
- おすすめ本タイトル: 海辺のカフカ
- 著者名: 村上春樹
- 手紙タイトル: 夜の終わりに渡したい一冊
- この本を選んだ理由: 読み終えたあとに残る静かな余韻を手渡したかったため
- 本の感想: 登場人物の孤独とつながりが、読後にゆっくり体温として残りました
- 相手に届けたい一言: あなたの夜が少しだけやわらかくなりますように
- ネタバレ: なし
- 気分タグ: 夜／余韻／静けさ

## TC-01 通常手紙を受信できる

目的:
通常手紙を開封し、手紙箱へ保存できることを確認する。

手順:
1. 画面上部の「受け取る」を開く。
2. 「受け取り名」に `夜の読書人A` を入力する。
3. 「この名前で受け取る」を押す。
4. 「おまかせで一通受け取る」を押す。

期待結果:
1. 通常手紙の本文が表示される。
2. 手紙内に差出人名が表示される。
3. 手紙詳細に「この人へ本の手紙を送る」ボタンが表示される。
4. 画面上部の「手紙箱」を開くと、受信した手紙が保存されている。

## TC-02 受信手紙から一冊の本を送れる

目的:
通常手紙から返信導線で「一冊の本の手紙」を作成し、保存できることを確認する。

手順:
1. 「手紙箱」を開く。
2. 直近で受信した通常手紙の「この人へ本の手紙を送る」を押す。
3. 表示されたフォーム「一冊の本を、手紙として送る」に次を入力する。
   - 差出人の名前: `夜の読書人A`
   - 相手の名前: `窓辺の受取人B`
   - おすすめしたい本のタイトル: `海辺のカフカ`
   - 著者名: `村上春樹`
   - 手紙のタイトル: `夜の終わりに渡したい一冊`
   - この本を選んだ理由: `読み終えたあとに残る静かな余韻を手渡したかったため`
   - 本の感想: `登場人物の孤独とつながりが、読後にゆっくり体温として残りました`
   - 相手に届けたい一言: `あなたの夜が少しだけやわらかくなりますように`
   - ネタバレ有無: `なし`
   - 気分タグ: `夜／余韻／静けさ`
4. 「一冊の本を送る」を押す。

期待結果:
1. 「あなたの一冊を、手紙として送りました」と表示される。
2. 元の通常手紙に「この人へ一冊送りました」が表示される。
3. 同じ通常手紙から重複して送る導線が出ない。

## TC-03 相手名で一冊の本の手紙を受け取れる

目的:
返信用の一冊手紙が、指定した相手名にのみ届くことを確認する。

手順:
1. 「受け取る」を開く。
2. 「受け取り名」に `窓辺の受取人B` を入力する。
3. 「この名前で受け取る」を押す。
4. 画面内のあなた宛て一覧に表示された一冊手紙の「この手紙を読む」を押す。

期待結果:
1. 「夜の読書人Aさんから、一冊の本の手紙が届きました。」が表示される。
2. 表示内容に次が含まれる。
   - 本
   - 著者
   - 手紙のタイトル
   - この本を選んだ理由
   - 感想
   - あなたへ
3. 「手紙箱」を開くと、一冊の本の手紙が保存される。

## TC-04 一冊の本の手紙がランダム受信や封筒棚に混ざらない

目的:
返信用手紙が直接配達のみであることを確認する。

手順:
1. 「封筒棚から一通選ぶ」を開く。
2. 一覧に返信用の一冊手紙が出ないことを確認する。
3. 「受け取る」に戻る。
4. 「おまかせで一通受け取る」を押す。

期待結果:
1. 封筒棚には通常手紙のみ表示される。
2. おまかせ便でも返信用の一冊手紙は開かれない。
3. 返信用の一冊手紙は「受け取り名」一致時の一覧でのみ表示される。

## TC-05 受け取り名が異なると一冊手紙を受信できない

目的:
宛先名による配達制御が効いていることを確認する。

手順:
1. 「受け取る」を開く。
2. 「受け取り名」に `別の名前C` を入力する。
3. 「この名前で受け取る」を押す。

期待結果:
1. あなた宛ての一冊一覧に「今は、あなた宛ての一冊の本の手紙は届いていません。」と表示される。

## TC-06 運営室で返信用手紙も確認できる

目的:
運営室で通常手紙と返信用手紙の両方が確認できることを確認する。

手順:
1. フッターの「運営室へ」を押す。
2. パスワード `obi-admin` でログインする。
3. 手紙一覧から返信用手紙カードを探す（種別表示: 一冊の本の手紙）。
4. 「詳細を開く」を押す。

期待結果:
1. 返信用手紙に次が表示される。
   - 差出人名
   - 宛先名
   - 元になった手紙ID
   - 本のタイトル
   - 著者名
   - 本文
   - ステータス
   - 投稿日時
2. 操作ボタンとして次が使える。
   - 目を通した
   - 配達できます
   - 封を閉じる
   - 相談対応済み

## 回帰チェック（最低限）

- 通常手紙の投稿は引き続き可能
- 通常手紙の受信・手紙箱保存・読後メモ保存は引き続き可能
- 相談/通報は通常手紙・返信用手紙の両方で可能
- 運営室ログイン/ログアウトは引き続き可能
